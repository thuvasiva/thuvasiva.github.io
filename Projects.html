Football Tilt Game

Description

Before coming to university, in my own time, I had a go at creating my own game on the Unity game engine. It gave me my first insights into game development and programming. Through the help of Unity Tutorials I managed to complete this project. The idea of the game is that the user tilts a ball through a series of posts on a virtual football pitch in the shortest time possible.

Stock Market Simulator

Description

As part of my object oriented programming module I created a stock market simulator application. The key objects in the simulation are Financial Instrument objects. This is a generalisation of the specialised objects Stock/Short,Cash and Bonds, all of which can be traded in the simulation. The aim is to make as much money as possible by buying and selling these financial instruments, each of which vary in a slight way. A picture of the application running is below.
